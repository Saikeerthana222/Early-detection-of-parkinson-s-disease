# Early Detection of Parkinson's Disease

## Overview

AI-powered web application for early detection of Parkinson's disease using deep learning and medical image analysis.

## Features

* Upload medical image
* AI prediction using trained model
* Result visualization
* Web-based interface
* Fast diagnosis support

## Tech Stack

* Frontend: React + Tailwind CSS
* Backend: Flask (Python)
* Model: TensorFlow / Keras
* ML Model: VGG16 Fine-tuned
* API: REST API

## Project Structure

```
backend/
frontend/
docs/
models/
```

## How to Run

### Backend

```
cd backend
pip install -r requirements.txt
python app.py
```

### Frontend

```
cd frontend
npm install
npm run dev
```
